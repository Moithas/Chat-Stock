## Setup Instructions

1. Download each image from the Discord CDN URLs (they may expire!)
2. Rename them according to the naming convention above
3. Push to your GitHub repository
4. Update `GITHUB_REPO` in `fight.js` with your actual repo path

The URLs will be: `https://raw.githubusercontent.com/YOUR_USERNAME/Chat-Stock/main/assets/fight/strike1.png`
